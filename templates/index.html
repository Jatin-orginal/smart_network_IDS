About the Problem

Cyber threats are increasing rapidly in modern networks. It is difficult to manually monitor suspicious traffic and identify risky activities. This project provides a beginner-friendly intrusion detection system to monitor packets, identify protocols, and generate alerts.

How It Works

1. Capture Network Traffic
2. Analyze Packets
3. Extract IP & MAC Address
4. Perform Risk Analysis
5. Generate Security Alerts
