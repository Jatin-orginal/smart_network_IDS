# Smart Network Intrusion Detection System

A beginner-friendly cybersecurity project developed using Flask, HTML, CSS, and JavaScript.

## Features
- Real-Time Packet Monitoring
- Phishing URL Detection
- MAC Address Detection
- Risk Classification
- Protocol Detection
- Live Dashboard UI

## Technologies Used
- Python Flask
- HTML
- CSS
- JavaScript
- Scapy

## Run Project

Install dependencies:

pip install -r requirements.txt

Run Flask server:

python app.py

Open browser:

http://127.0.0.1:5000
